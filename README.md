# AI Tools Installer for Linux

سكربت بايثون لتثبيت أدوات التطوير المدعومة بالذكاء الاصطناعي بضغطة زر واحدة.

## الأدوات المدعومة (Supported Tools)
1. **Amp Code CLI**
2. **Atlassian CLI (ACLI)** - يدعم `rovodev` بشكل كامل.
3. **Google Gemini CLI**
4. **Qwen Code CLI**

## طريقة التشغيل (Usage)

تأكد من وجود `Python 3`، ثم شغل الأمر التالي:

```bash
python3 install_ai_tools.py
